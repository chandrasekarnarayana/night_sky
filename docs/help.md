# Night Sky — Help & Tutorials

## Getting Started
- Use the location selector to set your observing site.
- Set the date/time (UTC by default) and click **Update**.
- Switch views: 2D (Rect/Dome) or 3D (OpenGL, if available).

## Exporting Charts
- Click **Export PNG** and choose size (2000–4000px recommended).
- Use label density and limiting magnitude to control clutter.
- Metadata and legend are included in exports.

## Custom Catalogs
- In Controls, choose **Catalog → Custom** and browse to your CSV (id,name,ra_deg,dec_deg,mag).
- You can also place custom plugins/catalog loaders in `~/.night_sky/plugins/`.

## Keyboard Shortcuts (core)
- Ctrl+N: Now
- Ctrl+R: Update
- Ctrl+E: Export
- Ctrl+L/P/C/D: Label toggles
- Ctrl+2/3: 2D/3D view

## Tips
- Enable high-accuracy ephemerides for better planet/Moon positions (downloads JPL DE421).
- Adjust light pollution (Bortle) and refraction toggles for realistic visibility.
- Use label density slider to reduce overlap.
